Tutorial:

In order to use application, enter the name of any pokemon(Spelled Correctly), OR enter it's ID number.
Basic details will be provided and the option will be given to learn it's moves that are able to be acquired or the games that said pokemon can be encountered in.
A new pokemon may be searched without reloading the page. Pokemon ID numbers range from (1-905).


Note: Some newer pokemon may not have moves or games listed in the API yet.
